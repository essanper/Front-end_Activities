Activity 2: Responsive design

The aim of this activity is to build a responsive layout using Bootstrap. Write the code under the following paragraph, it must have:

  • A menu bar with 3 options (Option 1, Option 2, Option 3), the last option must be a dropdown option with 2 options inside (Option 3.1, Option 3.2)
  • A red background main content space (8 cols, 12 cols on small) next to a blue background right bar (4 cols, 12 cols on small)
  • A grey background footer space full width
